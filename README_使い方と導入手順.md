## すぐに使う導入手順（おすすめ）
1. `00_運用ルール_文書管理ルール.md` を案件の運用ルールとして決める
2. `00_決定事項ログ.md` と `07_課題_論点管理票.md` を案件共通台帳として作る
3. 初回〜提案前は `01_初回ヒアリングシート.md` と `08_見積前提条件確認シート.md` を使う
4. 契約後は `09_キックオフ議事録.md` で体制とルールを揃える
5. 要件定義で `05_要件ヒアリングシート_機能要件.md` と `06_要件ヒアリングシート_非機能要件.md` を使う
6. 実装期は `10_週次定例議事録.md` を運用する
7. 変更は必ず `13_変更要求票_CR.md` を起票する
8. 受入〜リリースで `12_UAT受入確認シート.md` と `14_リリース判定記録.md` を使う
