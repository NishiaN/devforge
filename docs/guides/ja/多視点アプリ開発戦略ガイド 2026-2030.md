## 🏥 **1. ヘルスケア/医療 (HealthTech)**

### 📋 規制・コンプライアンス

**「生命に関わる最高水準の要求」**

- **医療機器プログラム（SaMD）対応**：薬機法/FDA 510(k)/CE-MDR準拠設計
- **医療情報標準**：HL7 FHIR R5/DICOM/IHE統合
- **個人健康情報保護**：HIPAA/個人情報保護法第2条3号（要配慮個人情報）
- **臨床試験データ管理**：GCP/21 CFR Part 11（電子記録・電子署名）
- **AI医療機器承認戦略**：IMDRF/PMDA/FDA AI/ML based SaMD Action Plan

### 🛠️ 推奨技術スタック

```
【バックエンド】
- 言語：Rust（安全性最優先）/ Go（高速処理）
- DB：PostgreSQL（監査証跡）+ TimescaleDB（バイタルデータ）
- メッセージング：HL7 FHIR Server（HAPI/Azure Health Data Services）
- 暗号化：AES-256-GCM + TLS 1.3（患者データ保護）

【フロントエンド】
- フレームワーク：React/Vue（医療従事者向けダッシュボード）
- モバイル：React Native（患者アプリ）
- アクセシビリティ：WCAG 2.2 AA必須（高齢者対応）

【AI/ML】
- 医療画像解析：PyTorch + MONAI（Medical Open Network for AI）
- 自然言語処理：BioBERT/ClinicalBERT（電子カルテ解析）
- 予測モデル：XGBoost + SHAP（説明可能性必須）

【インフラ】
- クラウド：AWS（HIPAA Eligible）/ Azure（健康データサービス）/ GCP（Healthcare API）
- コンテナ：Kubernetes + Istio（サービスメッシュ）
- 監査ログ：Splunk/ELK Stack（完全な追跡可能性）
```

### 🏗️ アーキテクチャ設計パターン

**電子カルテシステム（EMR/EHR）**

```
患者ポータル → API Gateway（認証・認可）
                  ↓
           FHIR Server（患者データ）
                  ↓
    ├─ 予約管理（Appointment）
    ├─ 診療記録（Observation）
    ├─ 処方箋（MedicationRequest）
    ├─ 検査結果（DiagnosticReport）
    └─ 医療画像（ImagingStudy → PACS連携）
                  ↓
         監査ログDB（変更履歴完全記録）
```

**リモート診療プラットフォーム**

- **WebRTC暗号化通信**：医師-患者間ビデオ診療（E2EE必須）
- **デジタル問診票**：構造化データ収集（FHIR Questionnaire）
- **オンライン決済**：医療費決済（Stripe Connect/GMO-PG）
- **処方箋電子化**：JAHIS処方データ交換規約準拠
- **同意管理**：FHIR Consent Resource（患者同意記録）

**AIによる診断支援**

- **画像診断AI**：肺がん/乳がん検出（FDA承認取得パス）
- **リスク予測**：糖尿病/心血管疾患発症予測
- **薬剤相互作用チェック**：処方提案最適化
- **バイアス監視**：人種/性別バイアス継続監視

### 🏗️ アーキテクチャ設計パターン

**デジタルバンキング**

```
【口座管理】

              ↓
         BFF（Backend for Frontend）
              ↓
    ┌─────────┴──────────┐
口座系API          勘定系API（既存システム）
    │                      │
    ├─ 残高照会         ├─ 入出金処理
    ├─ 取引履歴         ├─ 振込実行
    └─ 明細ダウンロード └─ 決済処理
              ↓
      Event Sourcing（全取引記録）
              ↓
    監査DB + Data Warehouse（規制報告）
```

**eKYC（本人確認）システム**

~~~python
class AdaptiveAuthEngine:
    def evaluate_transaction(self, tx):
        risk_score = 0
        
        # リスク要因加算
        if tx.amount > user.avg_transaction * 3:
            risk_score += 30  # 通常の3倍以上
        
        if tx.location != user.usual_location:
            risk_score += 25  # 異常な位置
        
        if tx.time.hour < 6 or tx.time.hour > 23:
            risk_score += 15  # 深夜時間帯
        
        if tx.device.id not in user.trusted_devices:
            risk_score += 20  # 新規デバイス
        
        # リスクベース認証
        if risk_score < 30:
            return "approve"  # 即座に承認
        elif risk_score < 60:
            return "sms_otp"  # SMS確認
        elif risk_score < 80:
            return "biometric"  # 生体認証
        else:
            return "block_and_call"  # ブロック＋本人確認電話
~~~

| **ハッシュ化**                 | パスワード保存   | 不可逆       | "password123" → "5f4dcc3b5..."                  |
| **マスキング**                 | 画面表示         | 元データ保持 | "1234-5678-9012-3456" → "***\*-\****-****-3456" |
| **汎化（Generalization）**     | 統計分析         | 精度低下     | "35歳" → "30-40歳"                              |
| **ノイズ追加**                 | 差分プライバシー | 統計的       | 年収500万 → 498万（±2%ノイズ）                  |
| **合成データ**                 | 機械学習訓練     | 完全人工     | GANで生成した架空顧客データ                     |

```json
def require_scope(required_scope):
    def decorator(f):
        @wraps(f)
        def decorated_function(*args, **kwargs):
            token = request.headers.get('Authorization').split()[1]
            payload = jwt.decode(token, PUBLIC_KEY, algorithms=['RS256'])
            
            if required_scope not in payload.get('scope', '').split():
                return jsonify({'error': 'Insufficient scope'}), 403
            
            return f(*args, **kwargs)
        return decorated_function
    return decorator
```

```javascript
// キーボードナビゲーション実装
class AccessibleMenu {
  constructor(menuElement) {
    this.menu = menuElement;
    this.menuItems = Array.from(
      menuElement.querySelectorAll('[role="menuitem"]')
    );
    this.currentIndex = 0;
    
    this.attachEventListeners();
  }
  
  attachEventListeners() {
    this.menu.addEventListener('keydown', (e) => {
      switch(e.key) {
        case 'ArrowDown':
          e.preventDefault();
          this.focusNext();
          break;
        case 'ArrowUp':
          e.preventDefault();
          this.focusPrevious();
          break;
        case 'Home':
          e.preventDefault();
          this.focusFirst();
          break;
        case 'End':
          e.preventDefault();
          this.focusLast();
          break;
        case 'Escape':
          this.closeMenu();
          break;
      }
    });
  }
  
  focusNext() {
    this.currentIndex = (this.currentIndex + 1) % this.menuItems.length;
    this.menuItems[this.currentIndex].focus();
  }
  
  focusPrevious() {
    this.currentIndex = 
      (this.currentIndex - 1 + this.menuItems.length) % this.menuItems.length;
    this.menuItems[this.currentIndex].focus();
  }
  
  // ... その他のメソッド
}

// フォーカストラップ（モーダル内）
class FocusTrap {
  constructor(element) {
    this.element = element;
    this.focusableElements = element.querySelectorAll(
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );
    this.firstFocusable = this.focusableElements[0];
    this.lastFocusable = 
      this.focusableElements[this.focusableElements.length - 1];
  }
  
  activate() {
    this.element.addEventListener('keydown', this.handleTab.bind(this));
    this.firstFocusable.focus();
  }
  
  handleTab(e) {
    if (e.key !== 'Tab') return;
    
    if (e.shiftKey) { // Shift + Tab
      if (document.activeElement === this.firstFocusable) {
        e.preventDefault();
        this.lastFocusable.focus();
      }
    } else { // Tab
      if (document.activeElement === this.lastFocusable) {
        e.preventDefault();
        this.firstFocusable.focus();
      }
    }
  }
}
```

~~~python
def find_duplicates_slow(items):
    duplicates = []

        if item in seen:
~~~
